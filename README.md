# [Challenge - Burger Shop 🔗](https://challenge-burger.vercel.app/)

## Instructions to Start

### First of all, Dependencies

```shell
npm install
```

### To Build

```shell
npm run build
```

### To Start Local

```shell
npm run dev
```

it will be able in [localhost:3000](http://localhost:3000)

## Development Requirements

- React ✅
- Typescript ✅
- ES6+ ✅
- React Hooks ✅
